# Maroon Blockchain Traceability

A blockchain-based supply chain traceability system for tracking product origins and certifications.

## Features

- **Product Certification**: Farmers can register and certify their products on the blockchain.
- **Verification**: Retailers can verify the authenticity of products.
- **Transparency**: Complete transparency of the product's journey from farm to retailer.
- **Role-based Access**: Different interfaces for farmers and retailers.
